# Installation
LUMA is a universal package manager built from scratch
```curl -fsSL https://mindyannakawee2-tech.github.io/luma/install.sh | bash```
Use this command to install LUMA via GitHub
or Install Locally using Release files
```
cd ~/Downloads
unzip luma_pkg_manager_v0.3.zip
cd luma_pkg_manager
sudo ./install.sh
```
Run this after downloading the release files
